INTRO

SVG (Scalable Vector Graphics) are an XML-based markup language for describing two-dimensional based vector graphics.

Coordinates and position

SVG uses a coordinate system similar to the one used by canvas. That is, the top left corner of the document is considered to be the point (0,0).

Example:
<rect x="0" y="0" width="100" height="100" />

<svg width="100" height="100">

<svg width="200" height="200" viewBox="0 0 100 100">

SVG animation with SMIL

    SMIL allows you to:
  • animate the numeric attributes of an element
  • animate transform attributes
  • follow a motion path

Animating attributes of an element

Add an <animate> element inside the attribute element, for example <rect>.

Animating the transform attributes

The <animateTransform> element let you animate transform attributes.

Animation following a path

The <animateMotion> element lets you animate an element position and rotation according to a path.
The path is defined the same way as in <path>.
